# Module: Opinion Packs

## Purpose

Opinion packs are optional presets that provide starter guidance without becoming architecture truth by default.

## Owns

- Preset/opinion-pack terminology.
- Proposed ADR behavior.
- Optional stack-aware memory guidance.
- Preset safety constraints.

## Does Not Own

- Accepted repository decisions.
- Organization standards.
- Core memory engine neutrality.

## Current Decision

The CLI term remains `preset`.

The architecture term is `opinion pack`.
