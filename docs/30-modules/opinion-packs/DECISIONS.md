# Opinion Packs Decisions

## P1.5: Preset Is The CLI Term

Use `preset` in CLI commands and user-facing help.

Use `opinion pack` in architecture docs to clarify the model.

## P1.5: Presets Propose

Presets can propose architecture choices but must not silently accept them.

## P4: Minimal Built-Ins First

P4 uses minimal built-in preset content to validate the opinion-pack contract.

Rich ecosystem guidance should be added later only when the model proves useful.
